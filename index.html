△

ত্রিভুজ (Triangle)

সংজ্ঞা: তিনটি রেখাংশ দ্বারা আবদ্ধ চিত্র।
বৈশিষ্ট্য: ৩টি কোণের সমষ্টি ১৮০°। ক্ষেত্রফল = ½ × ভূমি × উচ্চতা।

◻

বর্গক্ষেত্র (Square)

সংজ্ঞা: যে চতুর্ভুজের ৪টি বাহু সমান এবং কোণগুলো সমকোণ।
বৈশিষ্ট্য: কর্ণদ্বয় সমান। ক্ষেত্রফল = (বাহু)²

◻

আয়তক্ষেত্র (Rectangle)

সংজ্ঞা: বিপরীত বাহুগুলো সমান ও সমান্তরাল এবং কোণগুলো সমকোণ।
বৈশিষ্ট্য: ক্ষেত্রফল = দৈর্ঘ্য × প্রস্থ।

▱

সামান্তরিক (Parallelogram)

সংজ্ঞা: বিপরীত বাহুগুলো সমান্তরাল কিন্তু কোণ সমকোণ নয়।
বৈশিষ্ট্য: ক্ষেত্রফল = ভূমি × উচ্চতা।

◇

রম্বস (Rhombus)

সংজ্ঞা: ৪টি বাহুই সমান কিন্তু কোণ সমকোণ নয়।
বৈশিষ্ট্য: ক্ষেত্রফল = ½ × (কর্ণদ্বয়ের গুণফল)।

⏢

ট্রাপিজিয়াম (Trapezium)

সংজ্ঞা: যে চতুর্ভুজের একজোড়া বিপরীত বাহু সমান্তরাল।
বৈশিষ্ট্য: ক্ষেত্রফল = ½ × (সমান্তরাল বাহুদ্বয়ের যোগফল) × উচ্চতা।

◯

বৃত্ত (Circle)

সংজ্ঞা: একটি নির্দিষ্ট বিন্দু থেকে সমদূরত্ব বজায় রেখে আবদ্ধ বক্ররেখা।
বৈশিষ্ট্য: ক্ষেত্রফল = πr²। পরিধি = 2πr।

সকল সূত্র ও উপপাদ্য
১. পাটিগণিত (Arithmetic)
  • সরল মুনাফা: I = prn
  • চক্রবৃদ্ধি মুনাফা: C = p(1 + r)ⁿ
  • লাভ: বিক্রয়মূল্য - ক্রয়মূল্য
  • ক্ষতি: ক্রয়মূল্য - বিক্রয়মূল্য
  • শতকরা লাভ: (লাভ / ক্রয়মূল্য) × ১০০%
  • শতকরা ক্ষতি: (ক্ষতি / ক্রয়মূল্য) × ১০০%
  • ঐকিক নিয়ম: M₁D₁ = M₂D₂
  • গতি: v = d / t
  • গড়: রাশিগুলোর যোগফল / রাশির সংখ্যা
২. বীজগণিত (Algebra)
  • বর্গ: (a+b)² = a² + 2ab + b²
  • বর্গ: (a-b)² = a² - 2ab + b²
  • মান নির্ণয়: a² + b² = (a+b)² - 2ab = (a-b)² + 2ab
  • মান নির্ণয়: (a+b)² = (a-b)² + 4ab
  • মান নির্ণয়: (a-b)² = (a+b)² - 4ab
  • মান নির্ণয়: 2(a² + b²) = (a+b)² + (a-b)²
  • মান নির্ণয়: 4ab = (a+b)² - (a-b)²
  • মান নির্ণয়: ab = ((a+b)/2)² - ((a-b)/2)²
  • উৎপাদক: a² - b² = (a+b)(a-b)
  • উৎপাদক: (x+a)(x+b) = x² + (a+b)x + ab
  • ঘন: (a+b)³ = a³ + 3a²b + 3ab² + b³
  • ঘন: (a-b)³ = a³ - 3a²b + 3ab² - b³
  • ঘনের মান: a³ + b³ = (a+b)³ - 3ab(a+b)
  • ঘনের মান: a³ - b³ = (a-b)³ + 3ab(a-b)
  • ঘনের উৎপাদক: a³ + b³ = (a+b)(a² - ab + b²)
  • ঘনের উৎপাদক: a³ - b³ = (a-b)(a² + ab + b²)
  • সূচক: aᵐ × aⁿ = aᵐ⁺ⁿ
  • সূচক: aᵐ ÷ aⁿ = aᵐ⁻ⁿ
  • সূচকের ঘাত: (aᵐ)ⁿ = aᵐⁿ
  • লগারিদম: logₐ(MN) = logₐM + logₐN
  • লগারিদম: logₐ(M/N) = logₐM - logₐN
  • সমান্তর ধারা: a + (n-1)d
  • সমান্তর সমষ্টি: Sₙ = n/2 [2a + (n-1)d]
  • গুণোত্তর ধারা: arⁿ⁻¹
  • সেট: n(A ∪ B) = n(A) + n(B) - n(A ∩ B)
৩. জ্যামিতি (Geometry)
  • ত্রিভুজ ক্ষেত্রফল: ½ × ভূমি × উচ্চতা

৪. ত্রিকোণমিতি (Trigonometry)
